Repository organization:

  • Labs: Hands-on exercises in lectures
  • Lab 1: Foundations of the Web
    The lab consists of two segments: Web and HTTP Protocol, and Web Application Programming. In the Web and HTTP Protocol section, Wireshark is employed to analyze the synchronous HTTP protocol, comprising HTTP requests and responses. During this process, the browser initiates an HTTP request to the server and awaits the corresponding HTTP response, forming the fundamental communication cycle between the client and server in web interactions.
  • Lab 2: Front-end Web Development
    This Front End Development lab provided a comprehensive overview of HTML, JavaScript, Ajax, CSS, and the jQuery library. Part 1 involved designing an HTML page with tags and forms, integrating JavaScript in four ways. CSS, in the forms of inline, internal, and external styles, added elegance. jQuery facilitated AJAX calls to echo.php, and two web services for random jokes and age guessing were incorporated using jQuery Ajax and fetch.
